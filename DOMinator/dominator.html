Create New Element Remove Last Element Reset

This is an element creator in Javascript using DOM.
Fill in the boxes to create an element.
Choose what html tag you want in the tagName box and then put the class and id name in the correct spots.
Then in the innerHTML box, type whatever words you want displayed. Once you have done that, click on "Create Element" to create it

Click on the button "Remove Last Element" to remove the last element that was created

If you click on the "Reset" button, it will reset all of the inputs

If you click on the red "x" the element creator will disappear. To bring it back, click "Show Panel"
